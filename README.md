## Prérequis

Avant de commencer, assurez-vous d'avoir installé les éléments suivants sur votre machine :

- [Navigateur Web](https://www.google.com/chrome/) (Chrome, Firefox, etc.)
- Un éditeur de code (optionnel, mais recommandé) comme [Visual Studio Code](https://code.visualstudio.com/)

## Déploiement en Local

Pour déployer l'application en local, suivez ces étapes :

Ouvrez le fichier index.html dans votre navigateur. Vous pouvez le faire en double-cliquant sur le fichier ou en l'ouvrant via le menu de votre navigateur (Fichier > Ouvrir).

Vérifiez que toutes les ressources sont accessibles :

Assurez-vous que les images et les fichiers CSS sont correctement référencés. Les chemins d'accès dans le code doivent être relatifs ou absolus selon votre structure de dossier.
Testez les fonctionnalités :

Vérifiez que le diaporama fonctionne.
Testez les formulaires de connexion et de création d'événements pour vous assurer qu'ils s'affichent et fonctionnent comme prévu.

## Utilisation
Une fois l'application déployée, vous pouvez :

Naviguer à travers les événements e-sport à venir.
Créer de nouveaux événements si vous êtes connecté en tant qu'organisateur.
Vous connecter avec vos identifiants pour accéder à des fonctionnalités supplémentaires.
